# Legal Pages for Google Cloud Project

This repository hosts the public website for my Google Cloud project legal documents using GitHub Pages.

## Purpose

This site provides:
- Privacy Policy
- Terms of Service
- Basic project/contact information

## Published Pages

After GitHub Pages is enabled, the site will be available at:

- Home: `https://YOUR-USERNAME.github.io/`
- Privacy Policy: `https://YOUR-USERNAME.github.io/privacy.html`
- Terms of Service: `https://YOUR-USERNAME.github.io/terms.html`

If this is a project site instead of a user site, the URLs will include the repository name.

## Notes

These pages are intended to be publicly accessible for use with Google Cloud and related OAuth or app configuration requirements.
